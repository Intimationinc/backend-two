pip install grpcio grpcio-tools

Generate the Python gRPC code from the .proto file:
python -m grpc_tools.protoc -I=. --python_out=. --grpc_python_out=. add_service.proto
This generates two files: add_service_pb2.py and add_service_pb2_grpc.py.


How It Works:
The add_service.proto file defines the service and messages.
The JavaScript gRPC server listens for incoming gRPC requests and handles them by performing the addition.
The Python gRPC client connects to the server, sends a request with two numbers, and receives the sum from the server.
